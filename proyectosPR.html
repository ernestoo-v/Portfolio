Test TheOffice

Proyecto realizado con con la tecnología JavaFX, programado en IntelliJ IDEA. Consiste en un test de personalidad en el que el resultado es el personaje de la serie de televisión de TheOffice que más se acerque a ti.

Tenemos una base de datos con todas las preguntas y un sistema de puntuación personalizado para saber quién es el personaje que aparecerá al final

Contiene 3 tipos de preguntas, de 4 respuestas, de 2 respuestas y finalmente de 4 respuestas con imágenes

Las preguntas se recogen de manera aleatoria de la base datos gracias a un método creado.

Y dependiendo del tipo de pregunta carga la plantilla de 4 respuestas, de 2 respuestas o de imágenes

Gestión incidencias

Proyecto realizado en Java, usando una base de datos creada. El objetivo de esta práctica era combinar lo aprendido en java con los conocimientos de Mysql y Base de datos. Consiste en una gestión de incidencias en la que hay unos usuarios y un tipo de incidencias, los cuales introducen las incidencias que han sufrido. Permite visualizar las tablas, editar los campos, añadir nuevos, y finalmente eliminarlos. Tambíen existen una serie de consultas útiles para ver algunos datos

Permite ver cada una de las tablas que hay, en este caso estamos viendo la tabla de clientes

Podemos consultar los clientes por nombre o por email, en este caso por nombre
