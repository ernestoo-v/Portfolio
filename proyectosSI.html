Inicio de sesión

Proyecto realizado con con la tecnología SpringBoot, consiste en una gestión de usuarios gracias a una base de datos y el manejo de solicitudes http, junto a la creación de una API REST por el cual el cliente puede interactuar con la base de datos.

Utilizo Argon2 para la encriptación de la contraseña en la base de datos
